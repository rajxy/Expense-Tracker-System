#  Expense Tracker System (Python)

A modular expense management application featuring multi-parameter filtering, custom tagging, automated monthly budget summaries, and CSV export functionality.

##  Key Features
* **Categorized Expense Management:** Custom tags and multi-parameter transaction tracking.
* **Input Validation Workflows:** Built-in safeguards preventing negative numbers or malformed entries.
* **Monthly Analytics:** Automatic aggregation and visual category breakdown using Pandas.
* **Search & Export Engine:** Instant lookup by tag, category, or price range with CSV output capabilities.
